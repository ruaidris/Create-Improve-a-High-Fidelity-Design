Project Title
Study Productivity Mobile App

Project Description
This project focuses on designing a study productivity mobile application. The goal is to improve focus, task prioritization, and exam planning through a clear visual system and consistent UI components.

Project Stages

Low-fidelity wireframes created in Course 2 to define the core user flow and screen structure.
